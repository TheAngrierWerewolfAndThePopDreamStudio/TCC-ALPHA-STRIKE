SOBRE

Trabalho de Conclusão de Curso apresentado à Escola Técnica SENAI, unidade Celso Charuri de Curitiba, como requisito para obtenção do Título de Técnico em Desenvolvimento de Sistemas.

Projeto: ALPHA STRIKE

Desenvolvimento de um site/portal para uma empresa de airsoft em Curitiba, PR, com o objetivo de proporcionar não apenas informações e entretenimento, mas também agregar valor à experiência do cliente. O projeto se baseia nas metodologias Lean Office e DSDM, buscando oferecer uma solução ágil, adaptável e eficiente. O site oferece uma variedade de recursos, incluindo informações sobre airsoft, uma loja virtual, notícias, vídeos, reserva de arenas para jogos e treinamentos.

Projeto: ALPHA BLAST

Desenvolvimento de uma bomba em Arduino para jogos de airsoft, proposto como trabalho de disciplina de Internet das Coisas (IOT). O projeto teve como objetivo integrar dispositivos eletrônicos para adicionar complexidade aos jogos, oferecendo uma alternativa de baixo custo em comparação com as alternativas disponíveis no mercado. O objetivo principal foi desenvolver uma bomba utilizando conhecimentos adquiridos durante o curso sobre componentes eletrônicos e programação.
